Technical Details

An in depth explanation of the project and how AI can be combined with medicine.

Why are we doing this?

With current RCTs, we get an idea of how a vaccine/drug will work for the average person, but it can not give an idea for a specific person. Genetic Minorities are the ones most affected by this.

How do we know genetic minorities are affected and that genes play a part?

Refer this study on how minorities are affected in healthcare.

But Can Machine Learning predict such advanced things?

Yes, it can. Refer this study on how AI can predict the effect of vaccines on people.

What Machine Learning Models are being used?

Currently, we have the provision to use KNN, Linear Regression, Logistic Regression, Support Vector Classifier, Adaboost and Decision tree Regressor. Plans to add advanced neural networks are also in the process. Refer this study which recommends some of these models.

Alright, But how will we approach this?

First, we gather volunteers. Machine Learning always works better when there is more data available. Next, we collect their DNA data and test the drug/vaccine on them. We then map the DNA(not the whole but the parts corresponding to the drug's effect) with the drug/medicine's efficiency. Then we take a few genetic samples from various people and then test the Machine learning model and see if any results are bad/below the expected margin. We then find what exact genetic change causes this and develop a variation. We use this data to create a variation that performs better for the genetic minority. We repeat this process until we achieve the maximum precision possible for the most number of people.

What about the patients? Won't they get confused if there are so many variations?

To combat this problem an interface has been designed where a user can upload their genetic data and it shall automatically find the optimal variation for the person.
