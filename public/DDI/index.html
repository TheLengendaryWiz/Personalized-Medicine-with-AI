DNA Samples & Effeciency Data Input

Enter your DNA Samples as a CSV file. First, open up software like Google Sheets or Microsoft Excel. Then make two entries in the first row to make your title (I suggest to keep it as DNA and Effeciency but The title will be ignored). Next, enter all your inputs in the respective columns. Make sure all the DNA strings are of the same length. Effeciency must be a positive rational number. Next export it to a CSV file. To export to CSV , refer this for Ms-Excel and for Google sheets Click the File tab, Choose Download,Select the Comma-separated values (CSV) option.


Test Data

Enter a few sample dnas as diverse as possible sperated by a comma. It will then check and report the predictions for given set of dna. You can now easily find out which exact dna groups are at a disadvantage and start working on variations.
